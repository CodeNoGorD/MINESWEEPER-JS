LES REGLES DU JEU DEMINEUR

Le but du jeu est de trouver et de marquer les cases de la grille qui camouflent une mine. Lorsque vous cliquez sur une case, celle-ci vous indiquera le nombre de cases adjacentes contenant une mine. Si vous cliquez sur une case n'ayant aucune mine autour d'elle, celle-ci vous révélera automatiquement les cases adjacentes.

Vous pouvez marquer les cases que vous soupçonnez de cacher une mine en plaçant un marqueur "drapeau" dessus à l'aide du clic droit. Ceci désactivera toute possibilité de cliquer sur la case.

Vous remportez la partie lorsque vous parvenez à révéler toutes les cases sans mine. Si vous cliquez sur une case contenant une mine, la partie se terminera par une défaite.
